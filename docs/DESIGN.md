# Design direction

## Theme
Light theme. This is a newsroom tool used in bright offices and daylight travel environments. Clarity wins.

## Color strategy
Restrained. Tinted neutrals with one calm accent and a secondary signal color.

### Palette (OKLCH)
- Paper: oklch(0.98 0.01 85)
- Paper 2: oklch(0.95 0.012 85)
- Ink: oklch(0.2 0.02 250)
- Muted: oklch(0.55 0.02 250)
- Border: oklch(0.88 0.01 85)
- Accent: oklch(0.62 0.14 175)
- Accent deep: oklch(0.52 0.16 175)
- Signal: oklch(0.72 0.16 65)

## Typography
Use a single expressive sans. Prefer Sora for all UI text.

## Layout
- Wide, breathable spacing with a strong content rhythm.
- Avoid identical card grids. Use stacked panels and mixed blocks.
- Keep primary actions in the same place on each view.

## Components
- Buttons: solid, confident fills, gentle radius, visible active state.
- Panels: soft tinted surfaces with clear borders, no heavy shadows.
- Badges: simple, uppercase labels, no pill spam.

## Motion
- 150-200ms transitions.
- Ease-out only for enter/hover. No bounce.

## Copy style
- Short and direct.
- No em dashes.
